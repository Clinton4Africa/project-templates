RSS Readerテンプレート HTML+JavaScript 用
Version 1 (2012/10/16 版) 
Osamu Monoe 

このテンプレートを利用すると、RSS フィードを利用して記事のタイトル一覧と記事を表示できます。
以下の機能を搭載しています。
・アプリバー
・共有ソースコントラクト
・設定コントラクト
・検索コントラクト 
・スナップビュー/ポートレイトの対応
・コンテツンページ表示の際のスナップビュー解除
・オンライン/オフラインの判断


テンプレートを改造して使用する際のポイント:

[ロードする 情報の変更]
1) /js/data.js の中で、RSS の情報が定義されていますので、
目的の RSS の情報に合わせて書き換えます。

url: "http://www.microsoft.com/japan/msdn/rss.xml",
        group: {
            key: "MSDN",
            category: "developer",
            title: "MSDN オンライン"
	   }

内訳は以下の通りです。
 
url : RSS の URL
group.key : 未使用
group.category : フィルターに使用

RSS が一つしかない場合はソース内のコメントに従い操作してください。

2) アイテムリストのフィルターメニューの変更
2-1 以下のタグのメニューに表示されるキャプションを任意のものに書き換えます
<button id="filterMenu1"〜>デベロッパー</button>
<button id="filterMenu2"〜>IT プロ</button>

2-2  js/home.js の以下の関数を設定するフィルターに合わせて書き換えます。

[関数名]
setFilterMenu
[変更箇所]
execFilter 関数の引数 (※)
※data.js で定義している RSS の情報の group.category の内容です。

3) プライバシーポリシーの URL の変更
/pages/settings/privacy.js 内の変数 privacyURL にプライバシーポリシーを記載した Web ページの URL を記述します。

--------

注意事項:
・すべてのシナリオでテストしているわけではありませんので、予期せぬエラーの可能性があります。
・WACKテストは合格していますが、このテンプレートで作成したアプリがWindowsストアへ公開されることを保証するものではありません。
・取得したデータの永続化に対応していません。

【重要】
本テンプレートは第三者の RSS を推奨するものではありません。
RSS の二次利用には提供元の許諾が必要な場合があります。
アプリで使用する際には提供元に充分確認してください。
